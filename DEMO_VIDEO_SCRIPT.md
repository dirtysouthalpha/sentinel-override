# Sentinel Override - Demo Video Script

**Goal**: Show the agent planning, executing, and the cost safety features in action.

## Scene 1: Introduction
- **Visual**: Open the browser, click the Sentinel icon.
- **Audio**: "This is Sentinel Override, an open-source browser automation agent with hard-coded cost safeguards."

## Scene 2: The Plan-Execute Workflow
- **Action**: Type this prompt in the instruction box:
  > "Find the top 3 news headlines about AI from the BBC website."
- **Visual**: Show the **Plan** appearing (e.g., "1. Navigate to bbc.com, 2. Read headlines, 3. Summarize").
- **Action**: Click **Execute**.
- **Audio**: "The agent breaks the task into steps and waits for your approval."

## Scene 3: Execution & Shortcuts
- **Visual**: Watch the agent navigate and click automatically.
- **Action**: Once done, click the **Save** icon next to the prompt.
- **Audio**: "I can save complex prompts as shortcuts for future use."

## Scene 4: Cost Safety (The Sentinel)
- **Action**: Open **Settings** (Gear icon).
- **Visual**: Show the **Cost Safety** section (Budget: $5.00, Per-call limit).
- **Audio**: "And unlike other agents, Sentinel has hard-coded limits to prevent runaway costs."

## Scene 5: Conclusion
- **Visual**: Show the GitHub repository URL on screen.
- **Audio**: "It's free, open-source, and safer than the alternatives. Check it out on GitHub."
